14.1 Distribution for Population

Population <- sta$Population
summary(Population)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##    1.00   13.25   25.50   25.50   37.75   50.00
plot(Population)
hist(Population)
qqnorm(Population)
qqline(Population)
lag.plot(Population)
shapiro.test(Population)
## 
##  Shapiro-Wilk normality test
## 
## data:  Population
## W = 0.95558, p-value = 0.05809

Figure 14.1: Distribution Plot for Population

From the these plots we notice that the population data was scattered randomly, unevenly. The satter plot and histogram plot show that most states had population between 0 and 5000. The more the population, the less the frequecy. Some of states had much large population around 20,000. There are two big tails in Q-Q plot, which means the population for those states was depart from normality.The shapiro test gives p value as 1.906e-07, much away from 0.05. With 95% confidence We will reject the null hypothesis that the population was normally distributed.

14.9 Distribution for Region

Region <- sta$Region
counts <- table(Region)
counts
## Region
##  1  2  3  4 
##  9 16 12 13
pie(counts)
barplot(counts, ylab = "State Counts")
text(x=1:4, 1, paste("n=", counts))
counts <- sort(table(Region), decreasing = TRUE)
percentages <- 100 * counts / length(Region)
barplot(percentages, las = 3, ylab = "Percentage", col = "green")
text(x=1:4, 2, paste("n=", counts))

Figure 14.9: Distribution Plot for Region

From the pie charts, it looks like these 50 states divided relative into four different regions evenly. But the summary and the bar chart together tell us that we have relatively more states in South(16) and less states in Northeast(9). North Central and West have similar number of states(12 and 13).
